Delete a Worksheet

On the “Assignment” tab of the ribbon, there is a tool labeled “Delete Sheet”. It deletes the currently active sheet without prompting the use to confirm this irreversible action. However, if the user tries to delete the last remaining visible worksheet, the code will fail because each Excel workbook must always have at least one visible sheet.

Task #PointsTask Description
1 10 In the workbook named “deleteSheet-XX.xlsm,” you will find a module named Solution with a subprocedure named “deleteActiveSheet”. Using the “On Error” statement, add a line of code to the “deleteActiveSheet” sub procedure to prevent this error from happening.  That is, modify the code so when the user clicks the button and there is only one visible sheet, the sheet will not be deleted and the user will not be prompted with any message.
