Choosing a folder

In common need in VBA programming is to allow a user to select a file or folder upon which your code should operate. In this assignment, you will prompt the user to select a folder.

Task #PointsTask Description
1 10 In the workbook named "ChooseFolder-XX.xlsm," you will find a module named “Solution” with a sub procedure named "ChooseFolder”.  Your assignment is to complete this sub procedure so that when it is executed, it will prompt the user to select a folder from and write its full path to cell A1 of the active sheet.

You must use the "FileDialog" object discussed in the chapter of the course textbook that covers working with files and folders.

You need not worry if your code has an error if the user presses “Cancel.”  For this assignment, you can assume that the user will always make a valid folder selection and click “Ok.”
