Recording and Editing Macros

From time to time it is necessary to look across a set of numeric values and to identify which is the smallest (or the largest). In this assignment, you will complete a function to return the ordinal position of the smallest integer value in an array. The workbook you will use to complete the assignment has the function already created. This function already accepts a comma delimited list of values and builds an array holding the valid integer values in the order in which they were initially given. You only need to complete the final step which determines where the smallest integer is.
