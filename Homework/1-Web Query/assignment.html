Web Query

The Web Query Wizard is a tool that allows you to build a web query. In Excel, a web query can be configured to automatically request specific information from a web server. To start the Web Query Wizard, select the "Data" tab of the ribbon. Then in the left most group, select the "From Web" button. If you need more guidance on how to build a web query, Google is your friend.
