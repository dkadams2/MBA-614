ISBN stands for International Standard Book Number. It is the common identifier for books and is used around the world. The numbering scheme was first proposed for use in 1970. The original ISBN standard provided for 10 digits. In 2007, it was changed to use 13 digits.

Your function should allow spaces, hyphens, or periods to be used in the number.  In fact, any non-numeric characters should be ignored.  That is, your function should check to see if there are 13 numeric digits in the input.  If there are, your function should check to see that the 13 digits are a well-formed ISBN.

To determine if a 13-digit number is a valid ISBN, check to see if the last digit is the correct check digit for the number. To calculate the check digit of a 13-digit ISBN with number (87 12345 12345 1) follow the following six steps:

Step 1: add all digits in even position and multiply the result by 3: (7+2+4+1+3+5)*3=66.  (See the table below for clarification on which digits are even and which are odd.)

Step 2: add all digits in odd position. (8+1+3+5+2+4)=23.

Step 3: add the results of steps two and three: 66+23=89.

Step 4: Take the remainder when divided by 10: 89 divided by 10 = 8 with a remainder of 9.

Step 5: Subtract from 10: 10-9=1, so the check digit here is 1.

Step 6:  If the result of Step 5 is 10, then the check digit is 0.

Hint: If you are trying to tell if an integer is odd or even, recall that all odd integers when divided by 2 will have a remainder of 1.  Alternatively, you can write a For…Next loop that starts with either one or two and then steps by 2 to access odds and evens respectively.

Note: Be careful when using string values with the plus sign (+). If two strings are combined with the plus sign (+), the result will be concatenation, not addition. That is, while the expression 3+6 evaluates to 9, the expression "3" + "6" evaluates to "36".
